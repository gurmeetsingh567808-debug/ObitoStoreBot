# Obito Store Bot

A Telegram Filestore Bot with:

- Silent Batch Mode
- Only-Next-File Filestore
- Original Format Restore (Caption Included)
- Ordered Batch Restoration
- 1.5 Sec Delay for Safe Restore
- Admin System
- Deep Link File Access
- Auto Delete System
- No Inline Buttons
